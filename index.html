    먼저 제 포트폴리오 사이트에 방문해 주셔서 감사합니다.

    인터넷 없이 살 수 없는 이 한국에서 평소 웹이나 프로그램을 이용하며 발생하는
    작은 불편함과 궁금증, 그것을 해결하고 싶어 개발에 대한 호기심을 가져왔던 저는
    짧지만 유익했던 7개월 간의 연습과 준비 끝에 개발에 정글에 뛰어들고자 합니다.

    비록 짧은 기간 준비하여 미숙한 점이 많을 수 있으나, 노력과 성장 가능성만큼은
    다른 신입 개발자들과 비교해도 손색 없을 만큼 자신 있습니다 !

    또한 제 강점인 성실함과 꼼꼼함으로 깔끔한 UI,UX환경을 구축함과 동시에
    비 전공자만이 보여줄 수 있는 톡톡튀는 아이디어들로 Team과 함께
    예쁘게 돌아가는 톱니바퀴가 되고 싶습니다.

프로젝트-01 : 게시판홈페이지 :springedu

    [사용툴]:
  • java-STS
  • DB-orcle 11g
    [화면구성]:
  • Bootstrap
  • 웹표준
  • 오픈소스
    [사용기술]:
  • sql쿼리관리-mybatis,jdbctmplit
  • tomcatServer
  • junit5

Description
    처음 웹개발을 연습해 보려고 시작한 프로젝트입니다.
    가장 기본적인 게시판 사이트를 제작하였고, 회원의 C.R.U.D, 게시글과 댓글의 C.R.U.D기능까지 구현하였습니다.
    view부분은, 전체적으로 Bootstrap을 사용하여 디자인 했으며, 너무 심심한 부분은 오픈소스와 웹표준을 이용해 손봤습니다.
    내부는 sql문을 jdbctmplit과 mybatis를 둘다 연습해 보고 차이점을 알아보기 위해 두 가지를 동시 사용해 보았습니다.
    최종 구현 시 junit을 통한 검사 후 view부분과 연동하여 톰캣서버를 이용해 웹에서 사용여부가 잘 이루어지는지 확인하였습니다.

프로젝트-02 : 회사 홍보 및 인트라넷 사이트 :LKH

    [사용툴]:
  • java-STS
  • DB-orcle 11g
    [화면구성]:
  • Bootstrap
  • 웹표준
  • 오픈소스
  • Ajax
    [사용기술]:
  • sql쿼리관리-mybatis
  • Spring security
  • tomcatServer
  • junit5
  • 지도,주소API
  • FullCallendar

Description
    어느 아이티 회사로부터 의뢰를 통해 제작되었다는 가정 하에 만들어진 기업의 홍보 및 인트라넷 사이트입니다.
    팀프로젝트로 진행 되었으며, 저의 경우는 회원관리부분과 전체적인 view 부분의 일부를 맡아 작업하였습니다.
    view부분은 오픈소스와 웹표준을 이용해 깔끔한 ui환경을 제공하려 노력 하였고 배포해도 될 만큼의 퀄리티를 최대한 살리려 노력 했습니다.
    기업홈페이지라는 측면에서 보안부분을 중시하여 spring security를 이용해 비밀번화 암호화, 권한으로 페이지의 접근제한 등을 구현하였고.
    유지보수가 쉽도록 하기 위해서 mybatis를 이용해 쿼리를 관리하고, 주석을 달아 코드를 정리 하였습니다.
    기존의 타 기업의 사이트와 비교해도 밀리지 않는 퀄리티를 내기위해 디자인 등 디테일한 부분에 신경을 꽤 쓴 프로젝트입니다 .
